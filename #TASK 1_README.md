# 🖼️ Lumina — Image Gallery

A responsive image gallery built with pure HTML, CSS, and JavaScript as part of my Web Development Internship at **CodeAlpha**.

🔗 **Live Demo:** [darshanak11.github.io/CodeAlpha_Image_Gallery](https://darshanak11.github.io/CodeAlpha_Image_Gallery/)

---

## ✨ Features

- 📱 **Responsive Design** — works on all screen sizes
- 🗂️ **Category Filters** — Nature, Architecture, People, Abstract, Cityscape
- 🔍 **Lightbox Viewer** — full-screen image preview
- ⬅️➡️ **Next/Prev Navigation** — buttons + keyboard arrow keys
- 🎞️ **Filmstrip Thumbnails** — quick jump to any image
- 🖱️ **Hover Effects** — smooth zoom, overlay, and transitions
- 👆 **Touch/Swipe Support** — mobile friendly
- ⌨️ **Keyboard Support** — arrow keys + Esc to close

---

## 🛠️ Built With

- HTML5
- CSS3 (Grid, Flexbox, CSS Variables, Transitions)
- Vanilla JavaScript (no frameworks)

## 🚀 Getting Started

### Run Locally
1. Clone the repository
```bash
git clone https://github.com/Darshanak11/CodeAlpha_Image_Gallery.git
```
2. Open `Image_Gallery.html` in your browser

### Or use Live Server in VS Code
1. Install the **Live Server** extension
2. Right-click `Image_Gallery.html` → **Open with Live Server**
